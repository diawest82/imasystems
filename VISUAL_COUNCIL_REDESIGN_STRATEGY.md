### 2. TYPOGRAPHY REDESIGN

#### Current (Generic System Font)
```
Font: System fonts (-apple-system, BlinkMacSystemFont)
Hierarchy: Basic, minimal contrast
```

#### RECOMMENDED

**Headlines**: **Fraunces** or **Crimson Text**
- Serif for authority and trust
- Premium, elegant, distinctive
- Used for H1, H2, section headers

**Body Text**: **Inter** or **Outfit**
- Modern, clean, highly legible
- Excellent for reading
- Professional appearance

**Code/Technical**: **IBM Plex Mono**
- For security credentials, certificates
- Shows technical credibility

**Implementation**:
```css
/* Headlines - Authority */
font-family: 'Fraunces', serif;
font-weight: 600;
letter-spacing: -0.02em;

/* Body - Modern */
font-family: 'Inter', sans-serif;
font-weight: 400;
letter-spacing: 0;

/* Sizes (increased hierarchy) */
H1: 56px (up from 48px)
H2: 40px (up from 32px)
H3: 28px (up from 24px)
Body: 16px (keep)
```
